# E-commerce_TS
E-commerce system that uses typescript


## Setup - step by step

```bash
git clone https://github.com/hiruthstautland/organization_viewer.git

cd e-commerce_ts

npm run install
```

## Start
```
npm run start
```
For building
```
npm run build
```
